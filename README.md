### 게임 방식
---
![image](https://user-images.githubusercontent.com/57481424/114320228-f4fee800-9b4f-11eb-8fc4-c01f7e1abb9e.png)
- 3개 Life를 갖고 시작
- 랜덤으로 아이템을 생성하여 위에서 내려옵니다
- 펭귄을 화살표키로 컨트롤하여 오랫동안 살아남습니다!
- 시간이 지날 수록 투사체의 속도가 증가합니다
